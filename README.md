📌 Project Overview

This project predicts future salaries using historical data and machine learning models. It analyzes key factors like experience, education, and job role to estimate salary trends with high accuracy.

🚀 Features
📊 Data preprocessing & cleaning
📈 Exploratory Data Analysis (EDA)
🤖 Multiple ML models (Linear Regression, Decision Tree, Random Forest)
🎯 Model evaluation (MAE, MSE, R² Score)
🖥️ Simple user input interface for predictions
